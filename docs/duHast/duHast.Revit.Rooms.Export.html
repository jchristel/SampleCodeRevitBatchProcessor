duHast.Revit.Rooms.Export.to_data_room.populate_data_room_object(doc, revit_room)¶

Returns a custom room data objects populated with some data from the revit model room past in. data points: - room name, number, id - if exists: parameter value of SP_Room_Function_Number - level name and id (if not placed ‘no level’ and -1) - Design set and option :param doc: Current Revit model document. :type doc: Autodesk.Revit.DB.Document :param revit_room: The room. :type revit_room: Autodesk.Revit.DB.Architecture.Room :return: A room data instance. :rtype: DataRoom

duHast.Revit.Rooms.Export.to_data_room.get_all_room_data(doc)¶

Returns a list of room data objects for each room in the model. :param doc: Current Revit model document. :type doc: Autodesk.Revit.DB.Document :return: A list of room data instances. :rtype: list of DataRoom
