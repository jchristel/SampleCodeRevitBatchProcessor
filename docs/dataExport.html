DataExport.write_json_to_file(json_data, dataOutPutFileName)¶

Writes collected data to a new json formatted file.

Parameters

  • json_data (json object (dictionary)) – A dictionary to be written to file.

  • dataOutPutFileName (str) – Fully qualified file path to json data file.

Returns

Result class instance.

  • result.status. True if json data file was written successfully, otherwise False.

  • result.message will confirm path of json data file.

  • result.result empty list

On exception:

  • result.status (bool) will be False.

  • result.message will contain exception message.

  • result.result will be empty

Return type

Result
